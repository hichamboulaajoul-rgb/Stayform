# Stayform

**Votre accueil, zéro papier.**

Stayform est une application de check-in en ligne pour les hôtes Airbnb.
Envoyez un simple lien à vos voyageurs — ils sélectionnent leurs dates,
uploadent leur pièce d'identité et signent le contrat de location en ligne.
Un dossier PDF complet est généré automatiquement à la fin.

## Fonctionnalités

- 📅 Sélection des dates de séjour via calendrier interactif
- 👤 Formulaire d'identité du voyageur
- 📄 Upload de la pièce d'identité (CIN recto/verso, passeport)
- ✍️ Signature électronique du contrat de location
- 📥 Génération d'un dossier PDF complet (identité + contrat signé)
- 📱 Compatible mobile, tablette et desktop (iOS & Android)

## Utilisation

1. Hébergez le fichier `checkin.html` sur GitHub Pages, Netlify ou tout hébergeur
2. Partagez le lien à votre voyageur avant son arrivée
3. Le voyageur complète le check-in en 3 minutes
4. Téléchargez le PDF généré pour vos archives

## Technologies

HTML · CSS · JavaScript · jsPDF
